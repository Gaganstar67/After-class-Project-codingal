# ACP4 - My Media Adventure Project

## Media Files Added

This project contains the following media files:

### Images (Space Mission Gallery)
- **rocket.svg** - A colorful rocket with flames and stars
- **planet.svg** - A beautiful planet with rings and continents  
- **astronaut.svg** - An astronaut in a space suit exploring space

### Audio (Aquatic Beats Party)
- **music.wav** - Background music for the aquatic party section

### Video Poster
- **poster.svg** - Underwater themed poster with fish, jellyfish, and bubbles

### Video (To be added)
- **aqua.mp4** - You need to add an aquatic/underwater video file here

## How to Add the Video File

To complete the project, you need to add an aquatic/underwater video file named `aqua.mp4` to this folder. You can:

1. **Record your own video** - Record an underwater or aquatic-themed video
2. **Download from free sources**:
   - Pexels Videos: https://www.pexels.com/videos/
   - Pixabay Videos: https://pixabay.com/videos/
   - Free Video Stock: https://www.videvo.net/

3. **Use an online converter** - If you have a video in another format (MP4, AVI, MOV), convert it to MP4 format

4. **Place the file** - Save the video as `aqua.mp4` in the ACP4 folder
